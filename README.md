# Titanic Survival Prediction 

##  Project Overview
This project predicts whether a passenger survived the Titanic disaster using Machine Learning.

##  What I did
- Data Cleaning (handled missing values)
- Feature Engineering (Title, FamilySize, Alone)
- Encoding categorical data
- Model Training:
  - Logistic Regression
  - Random Forest

##  Results
- Accuracy: ~82%
- Cross Validation Score: ~0.82

##  Tech Used
- Python
- Pandas
- Scikit-learn

##  Outcome
Built an end-to-end ML pipeline from data preprocessing to model evaluation.
